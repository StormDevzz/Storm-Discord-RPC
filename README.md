# Storm Discord RPC - Android

🔒 **ЗАЩИЩЕННОЕ ПРИЛОЖЕНИЕ** - Только для использования, код закрыт лицензией

Android приложение для установки кастомного статуса в Discord Rich Presence с современным интерфейсом и системой уведомлений.

## ⚡ Особенности

- 🎨 **Современный интерфейс** - Material Design с фиолетовой темой
- 📱 **Управление статусом** - установка кастомного статуса Discord
- 🔄 **Фоновая работа** - стабильная работа через сервис
- 🔔 **Система уведомлений** - настраиваемые уведомления о статусе
- 📊 **Визуальный статус** - отображение состояния RPC в реальном времени
- 🛡️ **Защищенный код** - Read-Only лицензия

## 📱 Использование

1. **Установите приложение** - используйте предоставленный APK файл
2. **Введите Client ID** - при первом запуске введите ваш Discord Application ID
3. **Настройте статус** - введите основной статус, детали и состояние
4. **Запустите RPC** - нажмите кнопку "🚀 запустить" для активации
5. **Настройте уведомления** - включите уведомления в настройках приложения

## 🔧 Требования

- **Android 5.0 (API 21)** и выше
- **Разрешение на интернет** - для связи с Discord
- **Разрешение на фоновые сервисы** - для стабильной работы
- **Разрешение на уведомления** - для получения оповещений

## 📁 Структура проекта

- `MainActivityClean.java` - Основной экран с полным функционалом
- `DiscordRPCService.java` - Фоновый сервис для Discord RPC
- `StormNotificationManager.java` - Система уведомлений
- `ClientIDActivity.java` - Экран ввода Client ID
- `SettingsActivity.java` - Настройки уведомлений
